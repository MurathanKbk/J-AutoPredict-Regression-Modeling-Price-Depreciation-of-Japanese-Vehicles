# J-AutoPredict: Regression Modeling & Price Depreciation of Japanese Vehicles

## Motivation
Japanese automakers are globally renowned for their reliability and high resale value retention. However, the used car market still features significant price variance based on mileage, age, and trim level. This project aims to build an end-to-end data science pipeline that analyzes secondary market data for Japanese vehicles and applies regression techniques to predict fair listing prices. 

## Hypothesis
**The Reliability Premium:** The depreciation rate (price drop per year/mile) of top-tier reliable Japanese brands is significantly flatter over a 10-year period compared to the overall market average for standard vehicles.
